
SD card demo application
************************

scope:
   General Use

description:
   Application demonstrating the use of FAT filesystem to read/write
   files on SD card using the SPI interface

keywords:
   FAT Filesystem, SD, SPI

boards:
   XK-SKC-L2

This application demonstrates read/writes to SD card. At the end, it
prints the read/write performance of the FAT filesystem.
